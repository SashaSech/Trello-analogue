# Trello-analogue (Task Boards)

Учебный проект по преддипломной практике

## Реализация
- **Backend:** Laravel + Livewire  
- **Database:** MySQL  
- **UI:** готовая тема (Tailwind/Flux)

## Функциональность

### 1. Пользователи
- регистрация (логин/имя, email, пароль) с валидацией;
- авторизация существующих пользователей;
- после входа/регистрации — переход в профиль;
- редактирование данных пользователя и смена пароля.

### 2. Проекты
- список проектов пользователя;
- создание проекта (название, описание, логотип);
- редактирование проекта;
- удаление проекта.

### 3. Доска задач (Kanban)
- задачи по колонкам со статусами;
- добавление карточки (заголовок, описание, ответственный, проект);
- редактирование карточки;
- перенос карточки между статусами (drag & drop);
- удаление карточки.
